# Software Architectural Design Patterns Skill

A coding-agent handoff package for applying the 22 classic software design patterns across Creator/Creational, Structural, and Behavioral categories.

## Files

- `SKILL.md`: primary skill instructions, selection map, workflow, red flags.
- `PATTERN_CATALOG.md`: detailed cards for all 22 patterns.
- `DECISION_TEMPLATE.md`: reusable Pattern Decision Record template.
- `AGENT_HANDOFF_PROMPT.md`: short prompt to paste into another coding agent.
- `pattern_catalog.json`: machine-readable pattern metadata.
- `manifest.txt`: package file list.

## How to use

Give `SKILL.md` and `PATTERN_CATALOG.md` to the coding agent. For quick use, paste `AGENT_HANDOFF_PROMPT.md` and ask the agent to follow the Pattern Decision Record template.
